Overview:
---------
This is a simple quiz application developed in Python. The user is asked multiple-choice questions, and the score is calculated.

Features:
---------
- Randomized questions
- Score tracking
- JSON-based question storage
- Interactive and console-based

How to Run:
-----------
1. Install Python.
2. Place both 'quiz_app.py' and 'questions.json' in the same folder.
3. Run: python quiz_app.py
